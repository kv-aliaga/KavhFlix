# KavhFlix

#### Projeto: site estático pessoal de resenhas e recomendações de filmes
#### Autor: Davi (kv-aliaga)
#### Última atualização: 5 de novembro de 2025

## Descrição

KavhFlix é um site estático (HTML/CSS) que reúne recomendações de filmes, indicados ao Oscar, resenhas e uma página “Sobre Mim”. O conteúdo é focalizado em cinema, música e interesses pessoais do autor (filmes, Beatles, Pokémon, lugares, etc).
